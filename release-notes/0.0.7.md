| --- | ---: | ---: | ---: | --- |
| 12s | measured | measured | slower | Passed: `energy_similarity=1`, `rms_diff=0`, `peak_diff=0`, `equal_fraction=1`. |
| 60s | measured | measured | slower | Passed: `energy_similarity=1`, `rms_diff=0`, `peak_diff=0`, `equal_fraction=1`. |
| 60min | `978.3s` | `8425.913s` | `0.116x` | Passed: `energy_similarity=1`, `rms_diff=0`, `peak_diff=0`, `equal_fraction=1`. |

This release is correctness-first: decoded parity passed, long-media speed failed versus legacy CPU normalize.
